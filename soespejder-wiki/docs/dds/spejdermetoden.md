# Spejdermetoden – 7 elementer

Brug denne side som ramme for planlægning og debrief. Hvert element er koblet til konkrete aktiviteter på vandet.

## Patruljesystem
- Sejl i faste patruljer med klare roller.  
- Roter roller fra tur til tur for læring.

## Learning by doing
- Korte instruktioner → mange gentagelser på vandet.  
- Brug checklister og feedback i par.

## Friluftsliv
- Træn i realistiske forhold (vind, strøm, bølger).  
- Planlæg små ture med ansvar for sikkerhed og udstyr.

## Medbestemmelse og ansvar
- Lad patruljen selv vælge fokusøvelse og succeskriterier.  
- Debrief: Hvad virkede? Hvad prøver vi næste gang?

## Samfundsengagement
- Snak om hensyn på vandet, miljø og færdsel i havnen.  
- Deltag i lokale aktiviteter (strandrens, kapsejladser).

## Personlig udvikling
- Sæt individuelle mikro‑mål (fx “vende uden at stoppe”).  
- Fejr små fremskridt og giv specifik ros.

## Værdier (Spejderloven)
- Knyt adfærd på vandet til lovens punkter (se [Spejderloven](spejderloven.md)).

![Spejdermetoden oversigt](../assets/diagrams/spejdermetoden-oversigt.svg)

### Aldersspor
- 8–12 år: tydelige roller, simple mål, korte debriefs.  
- 12–16 år: selvledelse, planlægning, refleksionsspørgsmål pr. element.
