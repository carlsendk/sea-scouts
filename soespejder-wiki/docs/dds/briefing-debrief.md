# Briefing & debrief

Brug skabelonen for at skabe tryg, målrettet træning.

## Briefing (før sejlads)
- [ ] Mål: hvad vil vi træne i dag?  
- [ ] Område & vejr: vind, bølger, grænser.  
- [ ] Sikkerhed: roller ved [MOB](../ordbog.md#mob), kommunikation, [buddy](../ordbog.md#buddy)‑par.  
- [ ] Roller: ror, trimmere, bow/taktik (J/70).  
- [ ] Fokusøvelse: 10‑15 min blokke med feedback.

## Debrief (efter sejlads)
- [ ] Hvad lykkedes? Hvad var svært?  
- [ ] 1 ting hver tager med til næste gang.  
- [ ] Evt. foto/diagram til logbog.
