# Sikkerheds‑checklister

Printbare tjeklister til brug ved broen.

## Før sejlads (leder)
- [ ] Vejr (vind/bølger/sigt/temperatur) vurderet; go/no‑go  
- [ ] Følgebåd: kill‑cord test, brændstof, VHF  
- [ ] Roller aftalt (ror/trimmere/udkig) og [buddy‑par](../dds/buddy-system.md)  
- [ ] Område/rute og “kom‑hjem” signal aftalt  
- [ ] Vest/tøj tjekket (se [Personligt udstyr](personligt-udstyr.md))

## Under sejlads
- [ ] Radio‑disciplin og korte kald  
- [ ] Hold overblik og afstand; justér område ved pust/bølger  
- [ ] Husk pauser/varme i koldt vejr

## Efter sejlads
- [ ] Debrief: 1 ting der virkede, 1 ting til næste gang  
- [ ] Skader/slitage noteret; udstyr skyllet/tørret  
- [ ] Meld “alle hjemme”
