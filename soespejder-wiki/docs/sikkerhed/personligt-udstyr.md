# Personligt udstyr – detaljer

Det rigtige udstyr gør sejlads tryg og sjov. Brug tjeklisterne og justér efter vejr og vandtemperatur.

## Tjekliste (alle)
- [ ] [Redningsvest](../ordbog.md#redningsvest) i korrekt størrelse, spændt og lukket  
- [ ] Varme lag: uld inderst, fleece/uld mellem, vind‑/vandafvisende yderst  
- [ ] Hue/handsker når koldt (sommer: kasket/solbriller)  
- [ ] Sko der kan tåle vand og sidder fast (ikke klipklappere)  
- [ ] Lille kniv og fløjte  
- [ ] Drikkedunk og snack  
- [ ] Telefon i vandtæt pose (leder)  

## Kulde og vand
- Vand/luft < 10 °C: overvej [tørdragt](../ordbog.md#toerdragt) og ekstra lag.  
- Hav tæppe/håndklæde klar på broen til efter sejlads.  
- Aftal kortere pas og tættere pauser ved koldt vejr.

## Sol og varme
- Brug solcreme (reef‑safe hvis muligt) og drik vand i pauser.  
- Let vindjakke beskytter mod afkøling mellem øvelser.

## Aldersspor
=== "8–12 år"
- Brug visuel tjekliste og makker‑check før afgang.  
- Leder bekræfter vest/varme ved broen.

=== "12–16 år"
- Spejderne laver selv buddy‑tjek og pakker ekstra lag efter vejr.  
- Patruljefører krydstjekker hele holdet.

## Pak til tasken
- Skiftetøj i pose, ekstra trøje, lille snack.  
- Evt. brillerem til solbriller.
