# DDS Aktiviteter – guide

DDS har en officiel aktivitetsbank på dds.dk/aktiviteter med idéer og forløb, som vi kan bruge som inspiration til møder og ture. Indholdet er generelt spejderstof (lege, øvelser, refleksioner), som kan tilpasses sø‑kontekst.

## Hvor finder jeg den?

- Åbn i ny fane: https://dds.dk/aktiviteter  
- Brug filtrene for at ramme varighed, aktivitetstype og målgruppe.

## Filtre på dds.dk/aktiviteter

- Varighed: <15 minutter, 15–30 minutter, 30–60 minutter, 60–90 minutter, 90–120 minutter, 120+ minutter  
- Aktivitetstype: <10 deltagere, Drama og Musik, Evaluering og refleksion, Førstehjælp, Klar dig selv, Klima og Miljø, Knob, Kommunikation, Kreativitet og håndelag, Kultur og samfund, Leg, Madlavning, Mørke, Natur og friluftsliv, Orientering, Pioner og tovværk, Samarbejdsøvelser, Science, Sundhed, Sø og vand, Vandaktiviteter  
- Målgruppe: Familiespejder, Mikro, Mini, Junior, Trop, Senior

Tip: For søspejd er “Sø og vand”, “Vandaktiviteter”, “Knob”, “Kommunikation” og “Førstehjælp” ofte relevante – kombiner dem med målgruppe (8–12 / 12–16) og ønsket varighed.

## Sådan bruger du den i praksis

Planlægning af spejdermøde (30–60 min blokke):

1) Vælg fokus fra wiki’en (fx [MOB](cheatsheets/mob-checkliste.md), [VHF](vhf.md), [Sejlads](teori.md)).  
2) Find 1–2 relaterede aktiviteter via filtrene på dds.dk/aktiviteter.  
3) Tilpas til alder: 
   - 8–12 år: kort instruktion, flere gentagelser og lege.  
   - 12–16 år: mere selvledelse, scenarier og rollefordeling.  
4) Afslut med mini‑debrief: “Hvad virkede? Hvad prøver vi næste gang?”

På tur (2–3 timer dagsprogram):

- Brug aktivitetsbankens idéer som “land‑moduler” (fx knob, VHF‑rollespil, navigation på land).  
- Kombinér med vand‑moduler fra wiki’en (sejlads, MOB‑øvelser).  
- Husk sikkerhed: [Sikkerhed](sikkerhed/index.md), [Vejrgrænser](sikkerhed/vejrgraenser.md), buddy‑system.
