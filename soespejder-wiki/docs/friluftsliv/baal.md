# Bål

## Båltyper
- Pyramidebål – hurtig varme.  
- Pagodebål – meget glød.  
- Nødstarter – bål i vådt vejr.

## Optænding
- Tændstål, tændblokke eller vat.  
- Start med tynde kviste; byg gradvist større stykker på.

## Bålvett
- Bålplads/bålfad; slukningsmiddel (vand/spand) klar.  
- Forlad aldrig bål uden opsyn; sluk grundigt.  
- Ryd op og respekter naturen.

## Mad på bål
- Snobrød, pandekager, suppe; øv opskrifter og hygiejne.
