# Båd‑checklister (før/efter)

Print og brug ved broen. Marker med [x] når klar.

## Jolle/Optimist
- [ ] Vest på, [sværd](../ordbog.md#svaerd)/[ror](../ordbog.md#ror) monteret  
- [ ] [Fald](../ordbog.md#fald)/[skøde](../ordbog.md#skode) fri for knuder  
- [ ] Sejl hejses og trimmes let; bomsikring  
- [ ] Buddy‑tjek og ruteaftale

## J/70
- [ ] Fendere/fortøjninger klargjort, roller fordelt  
- [ ] Storsejl/fok klar; gennaker pakket korrekt  
- [ ] VHF/telefon, førstehjælp og kasteline  
- [ ] Kill‑cord i ledsagebåd, hvis relevant

## Svendborgjolle / Aalborgjolle
- [ ] Fendere og fortøjninger  
- [ ] Sejl/liner klare; åre/bådshage  
- [ ] Vejr/rute/roller aftalt

## Følgebåd (RIB/motor)
- [ ] Kill‑cord test, brændstof, lænsepumpe  
- [ ] VHF/telefon, kasteline, tæpper, førstehjælp  
- [ ] Område/vejrtjek, samlingspunkt aftalt

## Efter sejlads (alle)
- [ ] Skyl/tør sejl og udstyr; hæng vest/tøj til tørre  
- [ ] Tjek for skader/slitage; notér og meld  
- [ ] Rul liner pænt, oplad radio/lys
