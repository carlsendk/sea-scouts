# Aalborgjolle

Klassisk træjolle. Stabil og rummelig til begyndertræning.

## Grundrig
- Mast/bom, [storsejl](../ordbog.md#storsejl) og evt. [fok](../ordbog.md#fok).  
- Fendere og fortøjninger klar ved havn.

## Manøvrer
- Som jolle: vending/bomning, fokus på vægt og [fartkontrol](../sejlads/fartkontrol.md).

## Tjekliste før afgang
- [ ] Vest, VHF/telefon hos leder  
- [ ] Årer/bådshage, fendere, fortøjninger  
- [ ] Sejl/liner tjekket; ror/sværd ok  
- [ ] Vejr/rute/roller aftalt

## Vedligehold
- Skyl beslag, tør sejl, kontroller tovværk.
