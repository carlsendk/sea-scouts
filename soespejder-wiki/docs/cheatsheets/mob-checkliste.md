# MOB-checkliste (kort)

Brug denne checkliste som lommekort eller print. En PDF-udgave findes i `assets/cheatsheets/mob-checkliste.pdf`.

## Før afgang
- Brief besætning på roller ved MOB.
- Aftal nødkald: VHF kanal 16 eller telefon.
- Hav kasteline/bøje klar på dækket.

## Ved “mand over bord”
1. Råb “MOB!” og peg på personen.
2. Kast bøje/fender og hold visuelt kontakt.
3. Vend båden kontrolleret mod læ side af personen.
4. Stop fart (frigear / sejl blafrende) før optagning.
5. Hjælp personen op via stævn eller læ side, uden at ror/rig fanger linen.
6. Check for nedkøling/skader og meld status til leder.

## Aldersnoter
- **8–12 år:** fokus på råb/pegning og kasteline; leder styrer båden.
- **12–16 år:** kan tage ror, foretage VHF-opkald og lede optagning.
