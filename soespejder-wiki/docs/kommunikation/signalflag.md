# Signalflag

Signalflag (International Code of Signals, ICS) bruges til at kommunikere kort og tydeligt mellem fartøjer og fra land. På spejderniveau er det især nyttigt at kende enkelte flag og deres betydning.

## Vigtige enkeltflag
| Flag | Navn | Betydning |
|---|---|---|
| A | Alfa | Dykker nede – hold godt klar og sej langsomt |
| B | Bravo | Jeg tager om bord, losser eller har farligt gods |
| C | Charlie | Ja / bekræftelse |
| D | Delta | Hold klar af mig; jeg manøvrerer med besvær |
| E | Echo | Jeg ændrer kurs til styrbord |
| H | Hotel | Lodsen er om bord |
| N | November | Nej / afvisning |
| O | Oscar | Mand over bord |
| V | Victor | Jeg har brug for assistance |
| W | Whiskey | Jeg har brug for lægehjælp |
| X | X‑ray | Stop din hensigt og afvent mine signaler |
| Y | Yankee | Jeg driver for anker |

Note: Kombinationen “NC” (November + Charlie) anvendes som nødsignal (distress) i flagkode og betyder “Jeg er i nød; kræver øjeblikkelig assistance”.

## Brug i praksis
- I jolle/kølbåd er flag mest relevant til “A” (dykker), “O” (MOB) og nødsituations‑markeringer.  
- Brug flag fra land (bro/bådplads) til at signalere til søs, hvis radioen fejler.  
- Lær 3–5 flag og øv dem via små lege/quizzer.
