# Specialafmærkning

- Farve: gul.  
- Toptegn: kryds (X) – gult, hvis toptegn forekommer.  
- Lys: gult (valgfrit), særlige rytmer (fx Fl Y).  
- Betydning: særlige områder (fx skydeområder, rekreative zoner, opmærksomhedspunkter).  
- Passage: efter forhold – se søkort/noter/tavler.
