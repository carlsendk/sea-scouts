# Midtfarvandsafmærkning

- Farver: rød/hvid lodrette striber.  
- Toptegn: kugle.  
- Lys: hvidt, ofte Iso (isophase) eller LFl (lang blink).  
- Betydning: midt i farvandet/akse; indsejling.  
- Passage: begge sider.
