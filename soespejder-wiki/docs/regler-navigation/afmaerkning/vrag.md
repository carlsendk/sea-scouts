# Vragafmærkning

- Kan markeres med specielle vragafmærkningsbøjer (blå/gule lysmønstre) eller tavler.  
- Se lokale søkort/efterretninger for detaljer og passage.  
- Hold god afstand; følg anvisninger fra myndigheder.
