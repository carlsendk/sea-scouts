# Sejlskib – lanterner & figurer

- Under sejl (ingen maskindrift):  
  - Sidelanterner (rød/grøn) + agterlys (hvid).  
  - < 20 m kan vise samlet lanterne i mastetoppen (tricolor) (lokale regler kan variere).  
  - Optional: rødt over grønt (to mastelys) for “sejlskib” (ikke påkrævet).
- Dagfigur: ingen særlig (sejl er nok).  
- Hvis motor bruges (selv med sejl): vis som maskindrevet skib.
