# Maskindrevet skib – lanterner & figurer

- Under vej (fremdrift af maskine):  
  - Mastetoplys (hvid) + sidelanterner (rød/grøn) + agterlys (hvid).  
  - < 50 m: 1 mastetoplys; ≥ 50 m: 2 mastetoplys (anden agterud og højere).  
- Dagfigurer: ingen særlige (se specifikke fartøjer nedenfor).  
- Slæb/tog: se særskilt side for ekstra lys/figurer.
