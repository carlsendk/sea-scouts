# Ankerligger – lanterner & figurer

- For anker:  
  - En (1) hvid lanterne (all‑round) for fartøjer < 50 m.  
  - To hvide (foran og agter, højere for) for fartøjer ≥ 50 m.  
- Dagfigur: sort kugle.
- Aground: se særskilt side for ekstra røde lys/figurer.
