# Skib med dykker i vandet

- Flag: A‑flag (rigid replika) – blå/hvid.  
- Lys/tavler: som [uddybnings-/dykkerfartøj](uddybning-dykker.md) (RAM + spærret/åben side).  
- Sejl tæt forbi? Nej – hold god afstand og lav fart.
