# Klar dig selv

Kort: Bliv selvstændig på tur og til træning.

## Forløb 8–12 år
- [ ] Pak rygsæk til træning (tøj, vand, snack, regn).  
- [ ] Lav enkel madpakke og ryd op.  
- [ ] Knob: [pælestik](../ordbog.md#paelestik) og [ottetalsknob](../ordbog.md#ottetalsknob).

## Forløb 12–16 år
- [ ] Planlæg egen “mini‑tur”: mål, mad, oppakning.  
- [ ] Budget, indkøbsliste og ansvar for en aktivitet.  
- [ ] Hjælp yngre spejdere i patruljen.
