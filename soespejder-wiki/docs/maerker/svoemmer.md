# Svømmer

Kort: Vandvett, redningsvest og svømmeprøve efter alder/klubkrav.

## Forløb 8–12 år
- [ ] Vandvett & hypotermi (kort teori).  
- [ ] Øv flydeøvelser med vest.  
- [ ] Svømmeprøve i varmt vand/motionsbassin, ledsaget af instruktør.

## Forløb 12–16 år
- [ ] Svømmeprøve i åbent vand med sikkerhedsbåd.  
- [ ] Redningsteknik (kasteline, bugsering).  
- [ ] Førstehjælp efter vandkold (tørt tøj, varme).
