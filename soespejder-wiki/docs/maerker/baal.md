# Bål

Kort: Tænd sikkert bål og lav simpel mad.

## Forløb 8–12 år
- [ ] Tørre materialer, optænding og sikkerhed (vand/spand).  
- [ ] Tænd pyramidebål; sluk korrekt.  
- [ ] Popcorn eller snobrød.

## Forløb 12–16 år
- [ ] Båltyper (stjernebål, pagode, reflektor).  
- [ ] Mad over gløder (gryde/pande).  
- [ ] Affaldsminimering og efterlad‑ingen‑spor.
