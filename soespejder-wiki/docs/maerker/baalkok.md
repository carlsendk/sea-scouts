# Bålkok

Kort: Planlæg, tilbered og server et simpelt bålmåltid.

## Forløb 8–12 år
- [ ] Hygiejne, opgaver i patruljen, indkøbsliste.  
- [ ] Simpelt måltid (fx suppe eller pandebrød).

## Forløb 12–16 år
- [ ] Menu med to retter og efterlag; roller (kok, bål, opvask).  
- [ ] Tidsplan, råvarehåndtering, næringslære kort.
