# Kaptajn

Kort: Planlægning, navigation og ansvar for en kort tur.

## Forløb 8–12 år
- [ ] Vælg område og lav enkel rute A→B.  
- [ ] Tjek [sikkerhed](../sikkerhed.md) og vejr sammen.  
- [ ] Gennemfør tur i dagslys med følgebåd.

## Forløb 12–16 år
- [ ] Planlæg rute med [søkort](../ordbog.md#soekort) og [kompas](../ordbog.md#kompas).  
- [ ] Rollen som skipper: crew‑briefing, “go/no‑go”, ansvar.  
- [ ] Afslut med logbog og refleksion over beslutninger.

## Dokumentation og afslutning
- Plan (kort, vejr, grænser), logbog, roller.  
- Kort evaluering: hvad gjorde vi godt/sikkert?
