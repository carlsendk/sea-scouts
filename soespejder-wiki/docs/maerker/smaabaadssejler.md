# Småbådssejler

Kort: Lær rigging, basale manøvrer og sikkerhed i jolle/Optimist.

## Forløb 8–12 år
- [ ] Rigging med tjekliste (mast, [spryd](../ordbog.md#spryd), [fald](../ordbog.md#fald), [sværd](../ordbog.md#svaerd), [ror](../ordbog.md#ror), [skøde](../ordbog.md#skode)).  
- [ ] Sejlpunkter: [bidevind](../ordbog.md#bidevind), [halvvind](../ordbog.md#halvvind), [læns](../ordbog.md#laens) (med diagram).  
- [ ] Manøvrer: 3× vending, 3× bomning på kort bane.  
- [ ] MOB‑øvelse i følgebåd med kasteline.  
- [ ] Sikkerhed: vest, buddy‑par og “MOB!”‑råb.

## Forløb 12–16 år
- [ ] Selvhjulpen rigging + fejlretning (skøde/fald/spryd).  
- [ ] Fart vs. højde: trim‑lege og to‑bådsøvelser.  
- [ ] Kontrolleret kæntring/opretning i læ.  
- [ ] Rute A→B med enkel [pejling](../ordbog.md#pejling).  
- [ ] Log turen og kort debrief.

## Dokumentation og afslutning
- Logbog: dato, vejr, øvelser, læring.  
- Kort quiz om sikkerhed og sejlpunkter.  
- Foto af rigging eller manøvre.

## Sikkerhed
Se [Sikkerhed](../sikkerhed.md). Brug følgebåd og hold øje med vejrgrænser.
