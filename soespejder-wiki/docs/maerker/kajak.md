# Kajak

Kort: Basal padleteknik, manøvrer og sikkerhed.

## Forløb 8–12 år
- [ ] Vest, ind/ud‑stigning ved lavt vand/bro.  
- [ ] Fremadtag, bagtag, stop og drej.  
- [ ] Makkerkontakt og simple lege.

## Forløb 12–16 år
- [ ] Kantning, sideforflytning og styretag.  
- [ ] Redning: entring fra vandet eller makkerredning.  
- [ ] Kort tur med pause og naturfokus.
