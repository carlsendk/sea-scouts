# Gast

Kort: Teamwork ombord – roller, trim og manøvrer.

## Forløb 8–12 år
- [ ] Kende roller: ror, storsejl, fok, bow.  
- [ ] Kald: “Klar til vending/bomning – Klar! – Vender/Bommer!”.  
- [ ] Basistrim: storsejl ind/ud, fokskøde, vægtplacering.  
- [ ] 2× vending og 2× bomning på kort bane.

## Forløb 12–16 år
- [ ] Rolle‑rotation inkl. bow.  
- [ ] Sæt/bom/nedtag [gennaker](../ordbog.md#gennaker) på læns.  
- [ ] Træne taktik: kig op i banen, kald vindstød/skift.  
- [ ] Debrief: hvad gjorde fart, hvad gav kontrol?

## Dokumentation og afslutning
- Rollekort og foto af besætning i action.  
- Korte notater om trimvalg ved forskellig vind.

## Sikkerhed
Handsker, ryd cockpit, klar kommunikation. Se [Sikkerhed](../sikkerhed.md).
