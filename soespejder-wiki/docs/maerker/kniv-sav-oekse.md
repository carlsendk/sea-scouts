# Kniv, Sav & Økse

Kort: Sikker håndtering, skærpning og praktiske øvelser.

## Forløb 8–12 år
- [ ] Sikkerhedszoner og greb (kniv).  
- [ ] Save tynde grene, flække småklodser med kindøkse.  
- [ ] Pak og opbevar sikkert.

## Forløb 12–16 år
- [ ] Skærpning (bryne), vedligehold.  
- [ ] Effektive save/kløv‑teknikker og ergonomi.  
- [ ] Byg en simpel trefod eller bålrist.
