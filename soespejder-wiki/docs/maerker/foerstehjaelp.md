# Førstehjælp

Kort: Lær ABC, stop blødning og håndtér småskader.

## Forløb 8–12 år
- [ ] ABC (Airway, Breathing, Circulation) på dukke.  
- [ ] Lav forbinding på finger/underarm.  
- [ ] Hvornår ringer man 1‑1‑2?

## Forløb 12–16 år
- [ ] Scenarier: forstuvning, snitsår, slag mod hovedet.  
- [ ] ABC med stabilt sideleje og alarmering.  
- [ ] Hypotermi‑tjek efter koldt vand (tørre klæder/varme).  
- [ ] Sammensæt et lille førstehjælpskit til båden.
