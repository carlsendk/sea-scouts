# Kano

Kort: Basal padleteknik, manøvrer og sikkerhed.

## Forløb 8–12 år
- [ ] Vest, ind/ud‑stigning ved bro.  
- [ ] Fremadtag, bagtag, stop og drej.  
- [ ] Buddy‑afstand og simple lege.

## Forløb 12–16 år
- [ ] Sideskift, kantning og bagror.  
- [ ] Redning: T‑rescue eller bugsering.  
- [ ] Kort tur med pause og naturfokus.
